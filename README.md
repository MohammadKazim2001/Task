# Responsive Profile & Login Module

## Description

A React-based SPA with login and profile view, using localStorage for persistence and SCSS for styling.

## Features

- Login form with validation
- Profile view with stored data
- Light/Dark mode toggle
- Fully responsive design
- SPA (no page reload)

## Tech Stack

- React
- SCSS (no UI libraries)

## Run Locally

npm install
npm run dev
